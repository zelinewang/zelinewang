# Profile design studies

The canonical [GitHub profile](../README.md) uses an editorial hero and semantic Markdown so its evidence remains readable on mobile, searchable, and accessible. These studies explore three stronger visual metaphors without asking a visitor to choose the author's identity.

Each artwork is a dated render from public GitHub data. Project rows inside an SVG are illustrative, not clickable; the verified repository links remain native Markdown below each study.

## [Console](./console/)

An engineering-operator view: one continuous terminal session, restrained phosphor, explicit workflow gates, and ambient motion used only as a state signal.

**Best at:** technical process, developer-native personality, operational cadence.

## [Constellation](./constellation/)

A systems map: six public projects become named stars, with discovery notes, a spectrum of tools, and a deliberately quiet field.

**Best at:** systems thinking, creative technology, memorable visual identity.

## [Field Notes](./field-notes/)

An editorial research journal: dated public evidence, marginalia, a working inventory, and principles arranged like observations from the field.

**Best at:** thoughtful craft, recruiter readability, human tone.
